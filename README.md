# SDATTool
By FroggestSpirit

Unpack/Pack NDS SDAT Files

Make backups, this can overwrite files without confirmation

Usage: python3 SDATTool [SDAT File] [mode] [SDAT Folder] [flags]

Modes: 

      -u Unpack SDAT

      -b Build SDAT
